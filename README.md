# JavaScript Interview Practice

![JavaScript Banner](https://user-images.githubusercontent.com/68542775/167072911-dc31eac8-6885-4a05-9c25-279ecce22a79.png)

This repository contains **JavaScript interview questions with solutions**.  
It is primarily created for **daily practice** and to **document solutions for future reference**.

The focus is on writing **clean JavaScript code** and improving **logical problem-solving skills**.

---

## Purpose

- Practice JavaScript interview questions
- Improve problem-solving and coding clarity
- Build a personal reference for revision
- Help others who want JavaScript coding practice

---

## Topics Practiced

- Closures
- Currying and Infinite Currying
- Scope and Hoisting
- Event Loop and Async JavaScript
- Promises and Async/Await
- Arrays and String Problems
- Recursion
- Higher-Order Functions
- Polyfills (map, filter, reduce, bind, call, apply)
- Objects and Prototypes
- Debouncing and Throttling
- Shallow Copy vs Deep Copy
- Functional Programming in JavaScript

---

## Notes

- Solutions focus on **working code**
- Minimal or no explanation is included
- The goal is **practice and documentation**, not tutorials
